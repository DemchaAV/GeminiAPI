Сгенерируй список вопросов по данному образовательному уроку, охватывая все ключевые моменты темы. Особое внимание удели методам (methods).

Требования к вопросам:

Все методы должны быть указаны в полном виде с круглыми скобками:

Если метод не принимает параметры: methodName()
Если метод принимает параметры, обязательно указать параметры: methodName(param1, param2)
Вопросы должны включать следующие типы:

Описание метода → название метода
(Например: «Какой метод удаляет элемент по индексу?» Ответ: remove(int index).)
Название метода → описание действия метода
(Например: «Что делает метод clear()?» Ответ: Удаляет все элементы коллекции.)
Тип возвращаемого значения (return type)
(Например: «Какой return type у метода size()?» Ответ: int.)
Перегрузка методов (method overloading)
(Например: «Чем отличаются методы add(Object obj) и add(int index, Object obj)?»)
Разница между похожими по смыслу методами
(Например: «Чем отличается метод get() от метода peek()?»)
Область применения методов и структур данных
(Например: «В каких случаях предпочтительнее использовать LinkedList вместо ArrayList?»)
Вопросы могут быть разных типов:
Открытые вопросы
Вопросы с множественным выбором (multiple-choice)
Вопросы на соответствие (matching)
Правда или ложь (true/false)
Формируй вопросы таким образом, чтобы они:

Помогали закрепить материал
Развивали понимание различий и областей применения методов
